Seattle AirBnb for One

Material Boards ✦ Spec Sheet

First Interior Project
Seminole State College ✦ First Semester 2022

I created this project myself from scratch with the prompt of mid-century modern deli. I first chose the color palette and typography before sketching and refining the logo. I created patterns for the brand to use on stationery, product packaging, and the like. I created a mockup for the home page of their website in photoshop, placed it in the iPhone and iPad mockup, and created a magazine ad for the brand. I also decided to create a menu and mockup napkins to support the in-house diner aesthetic. Indesign, Illustrator, Photoshop
